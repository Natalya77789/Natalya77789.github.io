Галерея "Скриншоты проектов"

Навыки, приобретенные на курсе «Веб-дизайн и разработка сайтов»:

1. Взаимодействие с хостингами Timeweb.ru и Reg.ru.
2. Работа в онлайн-редакторе Figma (создание лендинг-страниц и многостраничных сайтов,
 с установкой связей между страницами).
3. Верстка сайтов с помощью HTML и CSS в среде visual Studio Code.
4. Разработка сайтов на основе CMS в конструкторе сайта Wordpress.
5. Программирование на языке JS (калькулятор).
6. Работа в облачном хостинге репозиториев Git.
7. Работа с базами данных посредством языка SQL (Описание базы данных сайта онлайн-магазина
и команд для управления БД, с созданием выборки для карточки товара).
8. Разработка сайтов на фреймворках react.js и vue.js.
9. Создание портфолио в виде многостраничного сайта.

Навыки, примененные при создании итоговой (дипломной) работы
«Многостраничный сайт-портфолио»:

1. Работа в онлайн-редакторе Figma при разработке элементов
 оформления сайта и макета страниц сайта.
2. Верстка сайта на языке гипертекстовой разметки HTML и языке
декорирования CSS с созданием зацикленной анимации в виде вертикального
слайдера-карусели (страница сайта «Дополнительно).
3. Разработка при помощи фреймворка react. На нем написана 2-я страница сайта
и прикреплена как приложение, на которое можно перейти
по изображению-ссылке на странице «Проекты»,
либо по иконке GitHub в футере любой страницы сайта.
4. Разработка на языке JavaScript – проект «Калькулятор» и «Галерея скриншотов проектов».
5. Размещение проекта в репозитории облачного хостинга Git.
